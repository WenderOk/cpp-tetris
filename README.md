# Тетрис на C++

Реализация классического тетриса в терминале, написанная на С++ в парадигме ООП. В дальнейшем появится больше механик и поддерживаемых платформ.

## Механики

- ✅ Игровое поле
- ✅ Блоки и их падение
- ✅ Удаление заполненных линий
- ✅ Поворот фигур
- ✅ Возможность проиграть
- ✅ Добавить очки
- ✅ Меню проиграша
- ✅ Сохранение рекорда
- ✅ Уровни сложности
- ✅ Цвета блоков
- ✅ Красивые меню
- ✅ Меню паузы

## Поддерживаемые платформы

- **Windows** 10/11
- **Linux** [Manjaro, Arch-based]
> [!NOTE]
> Для корректного отображения в Windows нужен эмулятор терминала с поддержкой кодировки UTF-8!
> Также возможны баги и мерцание при отрисовке в старых CMD и PowerShell
